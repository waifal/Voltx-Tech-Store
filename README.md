Voltx is a modern, responsive e-commerce website focused on technology products. From computers and components to gaming and smart home devices, Voltx provides a clean, user-friendly interface for browsing tech categories and finding the latest gadgets.

## Key Features

- **Tech-Focused Categories:** Organized sidebar for easy navigation through main tech topics.
- **Responsive Design:** Built with Bootstrap 5 to look great on desktop and mobile.
- **Modern UI:** Clean, compact cards, hover effects, and icons for intuitive browsing.
- **Powered by PHP:** Modular architecture for easy updates and expansion.
- **Front-End Technologies:** HTML5, CSS3, JavaScript, and Bootstrap for a polished look and feel.
